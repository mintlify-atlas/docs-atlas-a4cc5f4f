## Overview

`KanbanBoard` is a visual task management component using drag-and-drop functionality. It organizes tasks into three columns: To-Do, In Progress, and Done.

## Usage

```jsx
import KanbanBoard from './componentes/KanbanBoard';

<KanbanBoard
  proyectoId={project.id}
  equipo={projectTeam}
/>
```

## Props

<ParamField path="proyectoId" type="string" required>
  The project ID to load and manage tasks for.
</ParamField>

<ParamField path="equipo" type="array" default="[]">
  Array of team members who can be assigned to tasks.
  
  **Structure**:
  ```jsx
  [
    { email: 'user@example.com', rol: 'owner' },
    { email: 'collaborator@example.com', rol: 'editor' }
  ]
  ```
</ParamField>

## State Management

The component maintains several key states:

- **tareas**: Array of task objects
- **Modal states**: `isModalOpen`, `isEditModalOpen`
- **Form states**: `titulo`, `descripcion`, `fecha`, `prioridad`, `asignado`
- **Edit state**: `tareaEditando`

## Drag and Drop

Implemented using `@hello-pangea/dnd`:

```jsx
import { DragDropContext, Droppable, Draggable } from '@hello-pangea/dnd';

const onDragEnd = async (result) => {
  const { source, destination, draggableId } = result;
  
  // Validation
  if (!destination) return;
  if (source.droppableId === destination.droppableId && 
      source.index === destination.index) return;
  
  // Update local state
  const nuevasTareas = tareas.map(t => 
    t.id === draggableId 
      ? { ...t, estado: destination.droppableId } 
      : t
  );
  setTareas(nuevasTareas);
  
  // Persist to database
  await moverTarea(draggableId, destination.droppableId);
};
```

## Task Management

### Creating Tasks

```jsx
const handleCrear = async (e) => {
  e.preventDefault();
  if (!titulo) return;
  
  const asignadoFinal = asignado || "Sin asignar";
  
  await crearTarea(
    proyectoId, 
    titulo, 
    descripcion, 
    fecha, 
    prioridad, 
    asignadoFinal
  );
  
  // Reset form
  setTitulo("");
  setDescripcion("");
  setFecha("");
  setPrioridad("media");
  setAsignado("");
  setIsModalOpen(false);
  
  cargarTareas();
};
```

### Updating Tasks

```jsx
const handleActualizar = async (e) => {
  e.preventDefault();
  if (!tareaEditando || !tareaEditando.titulo) return;
  
  await actualizarTarea(tareaEditando.id, {
    titulo: tareaEditando.titulo,
    descripcion: tareaEditando.descripcion,
    fechaVencimiento: tareaEditando.fechaVencimiento,
    prioridad: tareaEditando.prioridad,
    asignadoA: tareaEditando.asignadoA
  });
  
  setIsEditModalOpen(false);
  setTareaEditando(null);
  cargarTareas();
};
```

## Task Properties

### Priority Levels

<ParamField path="prioridad" type="string" default="media">
  Task priority level:
  - `"alta"` - High priority (red indicator)
  - `"media"` - Medium priority (yellow indicator)
  - `"baja"` - Low priority (blue indicator)
</ParamField>

### Task Object Structure

```typescript
interface Task {
  id: string;
  titulo: string;
  descripcion?: string;
  estado: 'todo' | 'doing' | 'done';
  prioridad: 'alta' | 'media' | 'baja';
  fechaVencimiento?: string;
  asignadoA?: string;
}
```

```jsx
import KanbanBoard from '../KanbanBoard';

function ProjectsView({ user }) {
  const [proyectoActivo, setProyectoActivo] = useState(null);
  const [tabActivo, setTabActivo] = useState('kanban');
  
  // Build team array from project data
  const equipo = [
    { 
      email: proyectoActivo.ownerEmail, 
      rol: 'owner' 
    },
    ...(proyectoActivo.colaboradores || [])
  ];
  
  return (

      {tabActivo === 'kanban' && (
        <KanbanBoard
          proyectoId={proyectoActivo.id}
          equipo={equipo}
        />
      )}
    </div>
  );
}
```

## Helper Functions

### getNombreCorto

Extracts username from email:

```jsx
const getNombreCorto = (email) => 
  email ? email.split('@')[0] : '?';
```

## Loading Tasks

```jsx
const cargarTareas = async () => {
  const data = await obtenerTareas(proyectoId);
  setTareas(data);
};

// Load on mount and when project changes
useEffect(() => {
  cargarTareas();
}, [proyectoId]);
```

<Note>
Tasks are filtered by state in real-time. Moving a task updates the UI optimistically before persisting to the database.
</Note>

<Note>
The `equipo` prop must include the project owner with `rol: 'owner'` to properly display team roles in assignment dropdowns.
</Note>
