      ```javascript
      // Authentication is handled via Firebase
      import { auth, googleProvider, signInWithPopup } from './firebase';
      
      const result = await signInWithPopup(auth, googleProvider);
      const user = result.user;
      ```
    </Step>
    
    <Step title="Create your first project">
      Once you're logged in, navigate to the Projects view and create your first project. Projects help you organize related notes, tasks, and documents in one place.
      
      <Note>
        Projects support role-based permissions, allowing you to control who can view or edit content.
      </Note>
    </Step>
    
    <Step title="Start collaborating">
      Invite team members to your project by email. Choose their role (viewer or editor) and share the project link. All changes sync in real-time across all collaborators.
      
      <Tip>
        Use the built-in chat feature to discuss project details without leaving the platform.
      </Tip>
    </Step>
    
    <Step title="Organize with notes and tasks">
      Create notes with the rich text editor, organize them in folders, and track progress with Kanban boards. Everything stays synchronized automatically thanks to Firebase real-time updates.
    </Step>
  </Steps>
