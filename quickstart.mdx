Welcome to NoteMV! This guide will take you from account creation to creating your first collaborative project and note in under 5 minutes.

## Sign In with Google

NoteMV uses Firebase Authentication with Google Sign-In for secure, seamless access.

<Steps>
  <Step title="Click Continue">
    When you first visit NoteMV, you'll see an immersive 3D welcome scene. Click the **CONTINUAR** button to proceed.
    
    <Note>
      On your first visit, you'll see a development disclaimer. Accept the terms to continue.
    </Note>
  </Step>

  <Step title="Authenticate with Google">
    NoteMV will prompt you to sign in with your Google account. The authentication flow is handled securely through Firebase.

    ```javascript
    // Authentication is handled automatically
    import { auth, googleProvider, signInWithPopup } from './firebase';

    const result = await signInWithPopup(auth, googleProvider);
    // You're now authenticated!
    ```
  </Step>

  <Step title="Access Your Dashboard">
    After successful authentication, you'll be redirected to your personal dashboard where you can manage notes, projects, and snippets.
  </Step>
</Steps>

## Create Your First Project

Projects in NoteMV help you organize related notes and collaborate with your team.

<Steps>
  <Step title="Navigate to Projects">
    Click **📁 Mis Proyectos** in the left sidebar to access the projects view.
  </Step>

  <Step title="Create a New Project">
    Click the create button and fill in the project details:

    - **Name**: A descriptive title for your project
    - **Description**: Optional details about the project's purpose

    ```javascript
    // Projects are created in Firestore
    await addDoc(collection(db, "proyectos"), {
      userId: user.uid,
      ownerEmail: user.email,
      nombre: "My First Project",
      descripcion: "A collaborative workspace",
      fechaCreacion: new Date(),
      status: 'active',
      enPapelera: false,
      accesos: [user.email],
      colaboradores: []
    });
    ```
  </Step>

  <Step title="Invite Collaborators (Optional)">
    Share your project by adding collaborators:

    - Enter their email address
    - Choose their role: **editor** (can edit) or **visor** (read-only)
    - Click invite

    <Note>
      Collaborators will see the project appear in their dashboard automatically.
    </Note>
  </Step>
</Steps>

## Create Your First Note

Notes are the core of NoteMV. They support rich text editing, syntax highlighting, and real-time collaboration.

<Steps>
  <Step title="Open the Dashboard">
    From the main dashboard, click the **+ Nueva Entrada** button to create a note.
  </Step>

  <Step title="Add Content">
    In the note editor:

    1. **Title**: Enter a descriptive title
    2. **Tag**: Select a category (React, JS, Python, Git, etc.)
    3. **Content**: Write your note using the rich text editor

    The editor supports:
    - Bold, italic, underline, strikethrough
    - Headers (H1, H2)
    - Ordered and bulleted lists
    - Code blocks with syntax highlighting
    - Links

    ```javascript
    // Notes are saved to Firestore
    const docRef = await addDoc(collection(db, "notas"), {
      userId: user.uid,
      titulo: "My First Note",
      contenido: "<p>Hello, <strong>NoteMV</strong>!</p>",
      etiqueta: "General",
      proyectoId: null,
      carpetaId: null,
      cover: "",
      fecha: new Date(),
      enPapelera: false
    });
    ```
  </Step>

  <Step title="Save Your Note">
    Click **GUARDAR CAMBIOS 💾** to save your note. It will appear in your dashboard immediately.

    <Note>
      All changes are autosaved to Firebase in real-time.
    </Note>
  </Step>
</Steps>

## Organize Notes in Projects

Link notes to projects for better organization:

1. Create or open a note
2. Assign it to a project from the editor
3. Access all project notes from the project view

```javascript
// Associate a note with a project
await addDoc(collection(db, "notas"), {
  userId: user.uid,
  titulo: "Project Meeting Notes",
  contenido: "<p>Discussion points...</p>",
  proyectoId: "abc123",  // Link to project
  carpetaId: null,
  etiqueta: "General",
  fecha: new Date()
});
```

## Real-Time Collaboration

NoteMV supports real-time presence indicators when multiple users work on the same note.

<CodeGroup>
```javascript Presence Registration
// Your presence is registered automatically
import { registrarPresencia } from './servicios/notas';

await registrarPresencia(notaId, user);
// Other users see you're online
```

```javascript Live Updates
// Subscribe to note changes
import { suscribirNota } from './servicios/notas';

const unsubscribe = suscribirNota(notaId, (updatedNote) => {
  console.log('Note updated:', updatedNote);
  // UI updates automatically
});
```
</CodeGroup>
